# Likith Kumar — Engineering Portfolio

A clean, minimal, and futuristic engineering portfolio website built with HTML, CSS, and Vanilla JavaScript. Focused on Electric Vehicles, Control Systems, and Intelligent Mobility.

## 🚀 Live Demo
[View Live Portfolio](https://likithkumar.github.io/)

---

## 🛠️ Tech Stack
- **Structure**: Semantic HTML5
- **Styles**: Custom CSS (Modular Design System, BEM naming)
- **Logic**: Vanilla ES6+ JavaScript
- **Animations**: CSS Keyframes + Intersection Observer API
- **Design Inspiration**: Minimalist (Tesla/Apple style)

---

## 📂 Project Structure
- `/assets`: Images, UI icons, and fonts.
- `/css`: Modular stylesheets (tokens, base, components, layout).
- `/js`: Core modular logic and components.
- `/data`: Externalized JSON data for projects.

## 📦 Deployment to GitHub Pages

To deploy this portfolio to your own GitHub repository:

1. **Initialize Git**:
   ```bash
   git init
   git add .
   git commit -m "initial commit"
   ```

2. **Create a GitHub Repository**:
   - Create a new repository named `likithkumar.github.io` (or any name).
   - Add your remote: `git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git`

3. **Push to Main**:
   ```bash
   git branch -M main
   git push -u origin main
   ```

4. **Enable GitHub Pages**:
   - Go to **Settings** > **Pages** in your GitHub repository.
   - Under **Build and deployment**, select `Deploy from a branch`.
   - Select `main` branch and `/ (root)` folder.
   - Click **Save**.
