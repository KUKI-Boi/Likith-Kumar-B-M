## 🌌 About This Project

Welcome to the source code of my personal portfolio. Designed to reflect my expertise in **Intelligent Systems** and **Electric Mobility**, this platform bridges high-fidelity aesthetics with clean, module-driven architecture. 

The application utilizes **Three.js** and **Vanta.js** to render a dynamic, interactive globe backdrop, creating a premium feel reminiscent of advanced dashboard interfaces in the mobility sector.

### ✨ Key Features
- **Dynamic 3D Environments**: Animated particle globe backdrop rendering in real-time.
- **Component-Driven Vanilla JS**: Clean and modular JavaScript structure without heavy frameworks.
- **Responsive & Fluid Grid**: A robust CSS grid layout that adapts seamlessly across all devices.
- **Cinematic Reveals**: Micro-animations and scroll-reveals that make interactions feel alive.
